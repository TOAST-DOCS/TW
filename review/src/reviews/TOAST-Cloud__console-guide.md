# TOAST-Cloud / ko/console-guide.md — 자동 검수 결과

> Claude 자동 검수 초안입니다. 담당 라이터(TW)가 각 항목의 타당성을 검토합니다.
> 실제 원본 문서 수정은 이후 개발자가 수행합니다.

## 변경해야 할 항목
- [변경분] | 95행 | `설IP ACL 설정에 등록된 IP에서만` 문장 앞 오타 `설` 삽입 | `IP ACL 설정에 등록된 IP에서만`
- [기존] | 42·43·62·63·165·265·266·302·303행 | 이미지 `http://static.toastoven.net/...` | `https://`
- [기존] | 324행 표 | `[NHN Cloud](http://nhncloud.com)` | `https://nhncloud.com`
- [기존] | 329행 | `NHN Cloud> 회원 ID/비밀번호로 로그인` 깨진 `>` 제거 | `NHN Cloud 계정 ID/비밀번호`
- [기존] | 326행 | `NHN Cloud 회원으로` | `NHN Cloud 계정으로`(글로서리)
- [기존] | 335행 | `NHN Cloud 회원이 ... 다른 NHN Cloud 회원에게`(2건) | `NHN Cloud 계정`(글로서리)
- [기존] | 732행 | UI 경로 `NHN Cloud 고객 센터 [1:1 문의]` | `고객지원 > 문의하기`
- [기존] | 662·686행 | `발송 할 수 있습니다` | `발송할 수 있습니다`(붙여쓰기)
- [기존] | 662·686행 | `서비스 알림시에만` | `서비스 알림 시에만`(의존명사 `시`)

## 참고
- 등급: ⓐ 변경분 = 시급(R) / ⓑ 문서 전체 = 시급(R)
- ⓐ는 95행 `설` 삽입 표기 오류가 핵심. 113행 부근 빈 줄 이동 diff는 무의미한 공백 변경이라 조치 불필요.
- [참고] http:// 이미지·링크는 HTTPS로는 정상 도달 확인됨(죽은 링크 아님). 스킴 정정 차원의 수정.
- [참고] 732행 `고객지원 > 문의하기`는 콘솔 실측 권장(라이브 메뉴명 단정 불가).
- [참고] 342·343행 `회원 알림 수신 그룹`·`회원 목록`은 알림 수신 그룹 '회원'(별개 개념) → `계정` 변환 금지.
- [참고] 54·55·258·602·603행 `개인/사업자 회원`·`조직을 생성하는 회원`·`한국/일본 회원`은 약관·결제 주체 맥락 일반 '회원' → 일괄 변환 금지.
- [참고] 339행 `회원 계정`은 OWNER 권한 설명 맥락 → 건별 확인.
- [참고] 멤버↔계정 혼용(7·92·153행 등)은 기존 표기 다수 → 별도 일괄 정비 검토.
- [참고] 거버넌스 설정·알림 수신 그룹 관리 등 콘솔 메뉴/경로 다수 → 콘솔 실측 권장.
