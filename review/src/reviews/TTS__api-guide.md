# TTS / ko/api-guide.md — 자동 검수 결과

> Claude 자동 검수 초안입니다. 담당 라이터(TW)가 각 항목의 타당성을 검토합니다.
> 실제 원본 문서 수정은 이후 개발자가 수행합니다.

## 변경해야 할 항목
- [기존] | 49-51행 | 성공 응답 불릿 `*` | `-`
- [기존] | 54행 | 실패 응답 단일 항목 `*` | `-`

## 참고
- 등급: ⓐ 변경분 = 통과 / ⓑ 문서 전체 = 수정 필요(서식)
- 2차 검증: diff 2건은 모두 띄어쓰기·표기 개선으로 적절(행번호 확인). 기존 이슈는 응답 절 `*` 불릿뿐이며 용어/조사/링크/표·코드블록 이상 없음. 내부 상대 링크(`/nhncloud/...`)이고 통신사 고객센터·정착 영문 서비스명 등 오탐 가드 해당 없음.
