## 참고
- 등급: ⓐ 변경분 = 통과 / ⓑ 문서 전체 = 수정 필요
- 6월 diff는 영문 ID 앵커(`<a id>`) 삽입과 발신 프로필 설명의 '브랜드 메시지' 추가가 전부라 변경분 자체는 깨끗함(ⓐ=G).
- 전체 기준으로는 기존 오타(`가능`→`기능`), 어미(`확인 부탁드립니다`), 띄어쓰기(`바로가기`)가 남아 ⓑ=🟠. 기술 오류·링크 깨짐·콘솔 불일치는 없어 R 근거 없음.
- 행번호는 원문 콘텐츠 기준으로 재교정(초안은 prefetched txt 행번호를 사용했음).
