# Data-Lake-Storage / ko/release-notes.md — 자동 검수 결과

> Claude 자동 검수 초안입니다. 담당 라이터(TW)가 각 항목의 타당성을 검토합니다.
> 실제 원본 문서 수정은 이후 개발자가 수행합니다.

## 변경해야 할 항목
- [변경분] | 25행 | 불릿 기호 `*` | `-`
- [기존] | 31행 | 불릿 기호 `*` | `-`
- [참고] | 25행 | `GetBucketAcl`, `DeleteObjects`는 Amazon S3 호환 API명 | 영문 정착 표기 유지 정상
- [참고] | 25행 | 단일 자식 불릿 `추가됩니다` | 릴리스 노트 어미로 허용

## 참고
- 등급: ⓐ 변경분 = 수정 필요(O) / ⓑ 문서 전체 = 수정 필요(O)
- 6월 변경(2026. 06. 23. 신규 기능)은 H2 날짜/H3 카테고리 구조가 릴리스 노트 규칙에 부합. 불릿 기호가 `*`(룰셋은 `-`)인 서식 이슈만 존재하며 변경분(25행)·기존(31행) 동일. 기술 오류·링크·필수 섹션 누락·콘솔 UI 불일치 없음.
