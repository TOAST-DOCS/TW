# Monitoring-Cloud-Monitoring / ko/new-instance-metric.md — 자동 검수 결과

> Claude 자동 검수 초안입니다. 담당 라이터(TW)가 각 항목의 타당성을 검토합니다.
> 실제 원본 문서 수정은 이후 개발자가 수행합니다.

## 변경해야 할 항목
- [기존] | 211행 | 문서 제목 `## ...`이 H2(같은 H2인 `## 개요` 등과 계층 어긋남) | H1(`#`)로 변경
- [참고] | 372~407행 | GPU 표 리소스명 `GPU 성능`·`GPU 상태`·`GPU 클럭 이벤트` | 콘솔 실측 권장(단정 불가)

## 참고
- 등급: ⓐ 변경분 = 통과 / ⓑ 문서 전체 = 수정 필요(헤딩 계층 1건)
- 6월 diff(`구`→`기존` 통일, `(필요 시)` 교정, 점 목록 평문화, 표 단위 공백 제거, GPU 표 신규 추가)는 모두 품질 개선이라 변경분 지적 없음(ⓐ=G).
- ⓑ는 표 중심 문서라 용어/단위/헤딩/특수문자/링크/표 구조 스캔. `°C`·`µ`(U+00B5)는 단위 표기로 정상, EN/EM DASH 오용 없음. 제목이 H2로 시작하는 기존 헤딩 계층 이슈 1건만 존재(ⓑ=O).
- 232·250·273·303·313행 스크립트의 `http://169.254.169.231/...`은 link-local(169.254/16) 인스턴스 메타데이터 엔드포인트로 정상 — 깨진 외부 링크 아님(오탐 가드).
