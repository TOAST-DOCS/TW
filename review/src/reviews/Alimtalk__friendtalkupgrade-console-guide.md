# Alimtalk / ko/friendtalkupgrade-console-guide.md — 자동 검수 결과

> Claude 자동 검수 초안입니다. 담당 라이터(TW)가 각 항목의 타당성을 검토합니다.
> 실제 원본 문서 수정은 이후 개발자가 수행합니다.

## 변경해야 할 항목
- [변경분] | 354행 | 이미지 링크가 `(TODO: 콘솔 캡처 후 NHN static 호스트 URL 교체)` 플레이스홀더 | 실제 static URL로 교체(미교체 시 이미지 깨짐)
- [변경분] | 341행 | `권장/제한 사이즈`(368행 `최대 파일 크기`와 불일치) | `권장/제한 크기`
- [변경분] | 382행 | `SMS AppKey`(396행 `앱키`, 404행 `SMS 앱키`와 혼용) | 본문 표기 통일
- [기존] | 전체 | 점 목록 불릿 기호 `*` | `-`로 통일
- [참고] | 158·159·187·188행 | `푸시 알람` 표기 | `푸시 알림` 여부 콘솔 실측 권장(정착 표현이면 유지)
- [참고] | 359·360행 | 신규 동영상 섹션의 기술 수치(인코딩 5~10분/최대 3일, ERROR 자동 처리, 4GB/4시간/8K) | 동작·수치 사실이므로 작성자 확인 필요

## 참고
- 등급: ⓐ 변경분 = 시급(R) / ⓑ 문서 전체 = 시급(R)
- R 사유: 6월 추가된 동영상 관리 섹션(354행)에 `TODO` 플레이스홀더 이미지 링크가 그대로 배포되어 이미지가 깨짐.
- 나머지 6월 변경(발송합니다·비즈니스폼·조사·참고하세요)은 대부분 개선이며 별도 액션 불필요. 잔이슈는 `사이즈/크기` 불일치, `SMS AppKey` 표기 혼용, `*` 불릿.
