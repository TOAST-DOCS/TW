## 참고
- 등급: ⓐ 변경분 = 통과 / ⓑ 문서 전체 = 수정 필요(경미)
- 변경분은 빌드용 정렬 주석 1건과 섹션 앵커(`<a id>`) 6건 추가뿐으로 한글 본문·용어·기술 내용 변경이 없어 깨끗함(G).
- 전체(ⓑ)는 띄어쓰기 1건(`업로드할`)만 실질 🟠. 외부 깨진 링크·콘솔 경로·고객센터 표현 없음(이 문서는 API 레퍼런스). `미리보기`(299행) 붙여쓰기 정상. 예시 JSON의 `www.example.com` URL은 플레이스홀더로 검증 대상 아님.
